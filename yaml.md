# Minimal GitHub Portfolio with YAML

## Concept
Use a `portfolio.yaml` file in each selected repository to define only custom fields:

- `title`
- `description`
- `tags`

All other data (repo URL, stars, language, etc.) is fetched from GitHub.

---

## Example `portfolio.yaml`

```yaml
title: My Project
description: A short summary of the project
tags:
  - react
  - api
  - fullstack
